A plugin that displays the 'Total Minutes', 'Number of Logins'

== Description ==
A plugin that displays the 'Total Minutes', 'Number of Logins', and 'Average Time in Minutes Per Login' that each user has logged on the site. Stats can be exported to csv, and can be reset.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` 
2. Activate the plugin through the 'Plugins' menu in dashboard
